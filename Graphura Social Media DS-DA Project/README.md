# Graphura Social Media DS-DA Project

## Overview
A social media intelligence and influencer recommendation project combining multi-platform content analytics, machine learning, dashboarding, and an application layer for decision support.

## Core Project Assets
- `Graphura Social Media Content Intelligence.ipynb`: Instagram/content analytics workflow.
- `Graphura Influencer Collaboration system.ipynb`: influencer recommendation/modeling workflow.
- `Graphura Social Media Data.csv`: primary social media dataset.
- `Graphura-Influencer-Market-Analysis.pbix`: Power BI market dashboard.
- `app.py`: app-level interface and inference workflow.
- `best_model_pipeline.joblib`: trained model artifact.

## Structured Folders
- `data/raw`: synchronized source datasets.
- `notebooks`: standardized sequence + source notebook copies.
- `src`: reusable ML modules.
- `app`: app execution scripts.
- `models`: serialized model files.
- `dashboards`: BI artifacts.
- `reports`: report PDF and figure snapshots.

## Objective
Enable data-driven social content strategy and influencer collaboration decisions using engagement and performance signals.
