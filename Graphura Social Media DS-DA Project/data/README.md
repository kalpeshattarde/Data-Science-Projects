# Data

## Purpose
Stores all Graphura social media datasets and merged influencer inputs.

## Files
- `raw/Graphura Social Media Data.csv`
- `raw/Graphura Instagram Data.csv`
- `raw/Graphura LinkedIn Data.csv`
