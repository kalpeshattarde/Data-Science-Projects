# Source Code (`src`)

## Modules
- `preprocess.py`: load and standardize dataset fields.
- `feature_engineering.py`: derived features used by model and app.
- `train.py`: XGBoost-based training helper.
- `evaluate.py`: classification metrics.
- `predict.py`: batch prediction utility from serialized model.

## Alignment
These modules mirror notebook and app logic for repeatable training/inference.
