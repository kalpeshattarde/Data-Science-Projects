# Twarita NGO DA Project

## Overview
This repository captures data-analysis work for Twarita Sanstha program and financial reporting, including cleaned datasets, raw audit/operational documents, dashboard files, Excel analysis workbooks, and reporting outputs.

## Scope of Content
- `data/`: processed and source records used for analysis.
- `dashboard/`: Power BI dashboard for stakeholder insights.
- `Excel_Analysis/`: Excel-based financial and training analysis artifacts.
- `reports/`: presentation/report outputs for communication.
